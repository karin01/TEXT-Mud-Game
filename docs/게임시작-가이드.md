# NEON REQUIEM 2087 — 게임 시작 가이드

## 요약

- **게임 시작** = 프로젝트 루트의 `게임시작.bat` 더블클릭.
- **Node.js**·**Python**이 없으면 각각 자동 설치를 시도하고, 설치 후에는 **한 번 더** `게임시작.bat`을 실행하면 됨.
- **Flask 서버**는 `backend` 폴더에서 가상환경 생성 → `pip install -r requirements.txt` → 별도 창에서 실행됨.
- **프론트엔드**는 `npm install` 후 `npm run dev`로 Vite 개발 서버가 실행됨.

---

## 동작 순서

1. **Node.js 확인**  
   없으면 `winget install OpenJS.NodeJS.LTS` 후 터미널 재실행 안내.
2. **Python 확인**  
   없으면 `winget install Python.Python.3.12` 후 터미널 재실행 안내.
3. **Flask 백엔드**  
   - `backend\venv` 가상환경이 없으면 생성.  
   - `pip install -r requirements.txt` 로 Flask·flask-cors 설치.  
   - **새 창**에서 `python app.py` 실행 (포트 5000).
4. **프론트엔드**  
   - `frontend`에서 `npm install` → `npm run dev`  
   - 브라우저 자동 오픈 (Vite 기본 포트 5173).

---

## 사용 방법

| 방법 | 설명 |
|------|------|
| **게임시작.bat** | 더블클릭. Node·Python 확인 → Flask 설치/실행 → npm install → Vite 실행. |
| **게임시작.ps1** | PowerShell에서 동일 플로우. `.\게임시작.ps1` 또는 우클릭 → "PowerShell에서 실행". |

---

## 주의사항

- Node.js 또는 Python을 **방금 설치한 경우**에는 **터미널을 닫았다가** 다시 `게임시작.bat`을 실행해야 함. (PATH 반영)
- `winget`이 없거나 자동 설치가 실패하면 각각 [Node.js](https://nodejs.org/ko), [Python](https://www.python.org/downloads/)에서 수동 설치 후 다시 실행.

---

## 관련 파일

- `게임시작.bat` — 배치 파일 (기본 사용)
- `게임시작.ps1` — PowerShell 스크립트 (동일 로직)
- `backend/requirements.txt` — Flask, flask-cors
- `backend/app.py` — Flask 서버 진입점 (포트 5000, CORS 허용)
- `frontend/package.json` — `npm run dev` 로 Vite 개발 서버 실행
