## 2026-04-09 NPC 대화 분기(데이터 기반) 도입

- **목적/WHY**: NPC 대화 선택지가 `App.tsx`에 하드코딩으로 누적되면 유지보수가 어려워져, **대화 트리를 데이터로 분리**하고 공통 엔진으로 처리.
- **사용법**
  - `대화 <NPC이름>`: 대화 노드 출력 + 선택지 번호 표시
  - `선택 1`: 선택지 번호로 분기 선택 (기존처럼 키워드 입력도 일부 허용)
- **저장/영구 반영**
  - 분기 결과는 `playerState.story` / `playerState.npcs`에 반영되어 캐릭터 저장(휴식/세이브 등) 시 유지됨.
  - 대화 전용 플래그는 `story.dialogueFlags`에 저장.
- **구성**
  - `frontend/src/data/dialogues.ts`: NPC별 대화 트리 정의(샘플: 하나/카리나, 마이그레이션: 오니/고스트 퀸/리라)
  - `frontend/src/App.tsx`: 대화 세션 + 선택지 엔진(조건/효과/다음 노드) 적용
  - `frontend/src/utils/saveSystem.ts`: `dialogueFlags` 저장 스키마 확장(옵셔널)
