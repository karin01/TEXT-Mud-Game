## 2026-04-09 d20 전투 룰(명중/빗나감) + 잠금 보관함 도입

- **목적/WHY**: 전투의 명중/빗나감이 “운”처럼 보이지 않고, **주사위 결과 + 공격/방어 수치**로 설명 가능하도록 D&D 스타일 **d20 판정**을 도입.
- **규칙**
  - **d20 기본**: 1~20 굴림
  - **자연 20**: 치명타(자동 명중)
  - **자연 1**: 펌블(자동 실패)
  - **어드밴티지/디스어드밴티지**: 2번 굴려 **높은 값/낮은 값** 채택
- **명중률(연결식)**:
  - \(p = (ATK - DEF) / ATK\) 로 기본 성공률 산출
  - \(p\) 는 **5%~95%**로 클램프
  - \(p\) 를 **d20 DC(필요 최소 눈)** 로 변환해 판정
- **UI**
  - 명중/능력치 판정 시 **중앙 팝업 모달**로 결과(성공률/DC/d20)를 표시.
- **잠금 보관함(조사)**
  - 특정 방은 **INT 판정(DC)** 으로 열리는 보관함이 존재.
  - INT가 부족해도 STR이 충분하면 **부수기(STR 판정)** 로 강제 개방 시도 가능.
  - 부수기는 **무기 내구도 감소**가 발생할 수 있음(리스크/대가).
- **변경 파일**
  - `frontend/src/utils/d20.ts`
  - `frontend/src/App.tsx`
