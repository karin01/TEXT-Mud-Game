# NEON REQUIEM — 레전드리 등급 (에픽 상위, 드랍 전용) (2026-03-30)

## 요약

- `ItemGrade`에 **`legendary`** 추가. 사다리: `common` → … → `epic` → **`legendary`** (최종).
- 표기: **`[레전드리]`** (`ITEM_GRADE_LABEL`). 로그 색은 기존 `신화` ANSI(적색 계열)에 매핑.
- **상점 비판매**: `price: 0` → `getScaledShopBuyPrice` 미정, 아이언 잭 목록·구매 검증에서 `isShopExcludedGrade`로 제외. 베일 크립트 블라인드는 등급 상한이 에픽이라 **레전드리는 풀에 안 들어감** (`itemGradeLeq`).
- **드랍**: `items.ts`에 레전드리 베이스 6종 추가, `enemies.ts` 보스·미니보스 `lootPool`에 연결.

## 강화

- 에픽 티어 `+5` 다음 승급은 **레전드리 +0**.
- 레전드리 티어에서 `+` 올릴 때 재료 등급은 **에픽** (`getMaterialGradeForEnchantTarget('legendary')` → `'epic'`).
- 레전드리 `+5`에서 더 승급 없음.

## 주요 파일

- `frontend/src/data/items.ts` — 등급 타입·사다리·`itemGradeLeq`·`isShopExcludedGrade`·아이템 정의
- `frontend/src/data/ironJackShop.ts` — 목록 필터·상점 안내 문구
- `frontend/src/data/gamblingMerchant.ts` — `itemGradeLeq`로 상한 비교
- `frontend/src/data/enemies.ts` — 전리품 풀
- `frontend/src/App.tsx` — 강화 최종 티어 처리·일괄 판매 제외·구매 시 레전드리 안내
