# 장면 이미지 깨짐(깨진 아이콘) — 원인·수정 (2026-03-30)

## 원인

- Vite 기본 `base: '/'` 빌드는 번들 PNG URL을 `/assets/파일명` 형태로 넣는 경우가 있어, **문서가 사이트 루트가 아닐 때** 이미지 요청이 엉뚱한 곳으로 감.
- 예: Live Server로 `frontend`만 연 상태, `dist`의 상위 폴더를 루트로 연 경우, `file://`로 `index.html`만 연 경우 → 장면 `<img>` 404 → 깨진 그림 아이콘.

## 수정

1. **`frontend/vite.config.ts`** — `base: './'` 로 정적 자산·청크 URL을 **현재 HTML/JS 기준 상대 경로**로 생성 (`import.meta.url` 기반 해석 포함).
2. **`frontend/src/App.tsx`** — `sceneImage`가 빈 문자열일 때 `BG_ALLEY`로 넘기도록 방어 (`??`는 빈 문자열을 걸러내지 못함).

## 확인 방법

- `npm run build` 후 **`dist/index.html`이 있는 폴더**를 웹 루트로 열기 (또는 `npm run preview`).
- 장면 패널에서 계단/골목 배경·적 일러가 정상 로드되는지 확인.

## 휴식처 점령/오염도(원한) 시스템 (2026-03-30)

### 요구사항 반영

- **초기 허브 2곳(`clinic`, `slum_market`) 제외**: 나머지 안전지대는 몬스터가 진입하면 **함락(점령)** 가능.
- **함락 알림**: 로그창에 **ANSI 31(빨강)**으로 표시.
- **휴식 금지**: 함락된 안전지대에서 `휴식/세이브/캠프파이어` 시
  - `"몬스터의 악취와 소음 때문에 쉴 수 없습니다"` 메시지 출력.
- **미니맵(NAV-GRID)**: 함락된 안전거점 노드를 **붉은색**으로 표시(범례 포함).

### 오염도(원한) 추가

- **정의**: `mazePollution`
  - 미로/벌크 통로(예: `maze_*`, `bulk_*`, `silent_maze`)에서 몬스터를 처치할수록 증가
  - 증가량: 처치 수(중복 보상 가드 이후 unique defeat 기준)
- **효과**: 안전거점(초기 허브 제외) 이동 시 “습격/함락” 이벤트 확률 증가
  - 확률(현재): \(0.02 + 0.003 \times pollution\), 최대 0.35
  - 발생 시: 즉시 `villageOccupied[roomId]=true` + 1~2마리 전투 시작 + 빨간 경고 로그

### 확장 아이디어(방어 시설)

- **자동 터렛/방어막**: 골드를 써서 설치 → 습격 확률을 낮추는 방향으로 확장 가능
  - 예시: `safeHouseDefense[roomId] = { turretLv, shieldLv }`
  - 습격 확률에 \( -0.02 \times turretLv \) 같은 식으로 감산

- **목표**: 특정 NPC 일러가 없을 때도 장면 패널에서 **네온 레퀴엠 톤(네온/비/사이버펑크)**을 유지
- **조치**:
  - `frontend/src/assets/scene/images/npcs/npc_generic.png`를 새 폴백 이미지로 교체
  - 생성 스크립트: `frontend/scripts/gen_npc_generic_v2.py`
- **중요 (오해 방지)**: `mysterio.png` 등 **고디테일 NPC 원화는 프로젝트에 포함된 외부 원화 에셋**으로 보는 것이 맞다. Cursor 세션에서 Pillow로 “같은 수준으로 다시 그린 것”이 아니다.

## 미로 휴식처 등 배경이 안 보임 (어두운 프레임만) (2026-04-01)

- **증상**: 방 제목·로그는 `미로 휴식처`인데 장면 패널이 네이비 그라데이션만 보임.
- **원인 후보**: `import.meta.glob` 스캔이 일부 환경에서 비어 `pickAsset('backgrounds','bg_maze_rest')` 실패 → `<img>` 로드 실패 → data URI 폴백. + 장면 하단 그라데이션이 어두운 일러를 더 ‘비어 보이게’ 할 수 있음.
- **조치**:
  - `sceneImageUrls.ts`: `bg_alley`, `bg_stairs`, `bg_maze`, `bg_maze_rest`, `bg_extended_maze`, `bg_industrial_maze`를 **`?url` 정적 import** 후 glob에 키가 없을 때 맵에 주입.
  - `maze_rest` 분기: `pickFirst` 실패 시 **`bgMazeRestUrl` 직접 사용**.
  - `App.tsx`: 장면 패널 하단 그라데이션을 `from-[#0a0b10]/75`로 약화.
